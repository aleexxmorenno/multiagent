# 🤖 Multi-Agent Specialist Hub (ADK)

This repository demonstrates a **Multi-Agent Orchestration** system built with the **ADK (Agent Development Kit)**. The project features three specialized sub-agents and a shared **Tool** integration, allowing the system to handle diverse domain-specific queries within a single architecture.

## 🏗️ Architecture & Specialized Agents

The system acts as a multi-disciplinary assistant, routing tasks to the most qualified agent:

### 1. 📚 Book Recommender Agent
* **Role:** Literature and educational resource specialist.
* **Capabilities:** Provides personalized reading suggestions based on genre, author style, or specific learning goals.

### 2. 💰 Crypto Analysis Agent
* **Role:** Financial market analyst specialized in digital assets.
* **Capabilities:** Analyzes market trends, price action, and blockchain projects to provide data-driven insights into the crypto ecosystem.

### 3. 📰 News Analyst Agent
* **Role:** Real-time information and media specialist.
* **Capabilities:** Processes current events, summarizes news reports, and provides context on global headlines.

---

## 🛠️ Tool Integration
The agents are equipped with a **Shared Toolset**, allowing them to extend their capabilities beyond the static knowledge of the LLM. 
* **Tool Functionality:** google search.
* **Purpose:** Enables the agents to fetch live data, verify facts, or perform specific calculations autonomously.

## ⚙️ Technical Stack
* **Framework:** Google ADK (Agent Development Kit).
* **Architecture:** Multi-Agent Routing & Tool-Use.
* **Language:** Python.
* **AI Integration:** LLMs with function-calling capabilities.
